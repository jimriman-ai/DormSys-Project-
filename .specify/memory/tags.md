# Research Memory Taxonomy

Allowed Tags:

- WRONG_PATH: An investigation path that proved unproductive.
- DEAD_END: A technical blocker preventing progress.
- LOOP_DETECTED: A recurring cycle in decision/review.
- AMBIGUITY: Unresolved definition in requirements or scope.
- EVIDENCE_GAP: Missing data needed for decision-making.
- EVIDENCE_CONFLICT: Contradictory inputs/facts encountered.
- OWNER_DECISION_NEEDED: Blocker requiring external authority.
- CHANGES_REQUIRED: Review-driven modifications.
- DEFERRED: Item postponed; not for this cycle.
- CLOSED: Final disposition.
- SAFE_PROGRESS: A pattern that proved resilient/successful.

Rules:

- Use only defined tags.
- Max 3 tags per log.
- Do not create new tags without Governance approval.
